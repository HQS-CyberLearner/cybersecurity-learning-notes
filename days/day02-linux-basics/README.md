# Day 2: Linux基础与Git入门

## 学习内容
1. **15个核心Linux命令**：掌握了文件操作、目录导航、权限管理等
2. **第一个Bash脚本**：编写了系统信息收集脚本
3. **Git基础操作**：学会了clone、add、commit、push

## 文件说明
- `linux_commands_cheatsheet.md` - 命令速查表
- `system_info_collector.sh` - 系统信息收集脚本
- `system_report_*.txt` - 脚本生成的报告
- `practice_commands.txt` - 练习记录

## 关键收获
1. Linux命令的熟练度通过实践提高
2. 脚本可以自动化重复任务
3. Git是版本控制和协作的必备工具

## 遇到的问题
1. 权限问题：使用chmod +x添加执行权限
2. Git配置：需要正确设置用户名和邮箱
