# Linux命令速查表 - Day 2

## 目录与文件操作

### 1. 导航
- `pwd` - 显示当前目录
- `ls [选项]` - 列出目录内容
  - `-l`：详细列表
  - `-a`：显示所有文件（包括隐藏文件）
  - `-h`：人类可读的大小
- `cd [目录]` - 切换目录
  - `cd ~`：回家目录
  - `cd ..`：上一级目录
  - `cd -`：上一个目录

### 2. 创建与删除
- `mkdir 目录名` - 创建目录
  - `-p`：创建多级目录
- `touch 文件名` - 创建空文件
- `rm [选项] 文件` - 删除
  - `-r`：递归删除目录
  - `-f`：强制删除
- `rmdir 目录名` - 删除空目录

### 3. 复制与移动
- `cp [选项] 源文件 目标` - 复制
  - `-r`：复制目录
- `mv 源文件 目标` - 移动/重命名

### 4. 查看内容
- `cat 文件` - 显示全部内容
- `less 文件` - 分页查看（空格下一页，q退出）
- `head -n 文件` - 显示前n行
- `tail -n 文件` - 显示后n行
- `nano 文件` - 简单文本编辑器

### 5. 权限管理
- `chmod 权限 文件` - 修改权限
  - 数字模式：755 (rwxr-xr-x)
  - 符号模式：+x (添加执行权限)
- `chown 用户:组 文件` - 修改所有者

## 查找与信息

### 6. 搜索
- `find 路径 -name "模式"` - 查找文件
- `grep "文本" 文件` - 在文件中搜索文本
  - `-r`：递归搜索
  - `-i`：忽略大小写

### 7. 系统信息
- `uname -a` - 系统信息
- `whoami` - 当前用户
- `df -h` - 磁盘使用
- `free -h` - 内存使用
- `ps aux` - 查看进程

## 实用技巧

1. **Tab键自动补全**：输入部分命令/文件名后按Tab
2. **上箭头**：调出上一条命令
3. `Ctrl+C`：终止当前命令
4. `Ctrl+L`：清屏
5. `man 命令`：查看命令手册（如 `man ls`）
